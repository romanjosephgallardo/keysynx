# KeySynx

KeySynx is a community-driven music metadata platform that allows users to browse, validate, and contribute information about a song's musical key, BPM, and song structure. It was developed as a final project for the Web and Mobile Systems course.

---

## Features

- Browse a database of 400+ songs
- Search by artist, title, musical key, and BPM
- Harmonic Wheel for compatible key visualization
- Community voting and comments
- Song submission workflow
- Email verification
- Reputation system
- Admin moderation panel
- Artist artwork management
- Responsive interface with light and dark mode
